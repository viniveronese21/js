# Queue

* Tradução de Queue é fila.

Como uma fila em uma loja ou restaurante

*Linear

* Oprimeiro que entrar é o primeiro a sair 

# Conceitos

* FIFO: Fist In First Out
    * Oprimeiro elemento a entrar nafila, é o primeiro elemento a sair dela .

* Front (Frente) é a referencia do primeiro a entrar na fila

* Back (Fundo) é a fererencia do ultimo elemento a entrar na fila

# Queue no código 

* Métodos Fundamentais 
    * enqueue(): adicionar um elemento ao final da fila
    * dequeue(); remover o primeiro elemetno a entrar na fila

* Outros métodos poderão ser implementados como 'size()' para mostrar o tamano da fila ou 'front()' para pegar o peimeiro elemento da fila, dentre tantos otros
