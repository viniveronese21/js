# O que é estrutura de dados? 

* Uma maneira de organizar e ordenar informações como textos, números, booleanos, etc e registá-los na memoria do computador

* Organizar dados ( informações )   
    
    * Textos, números e booleanos ...

* Como estão registrados na memória

    * Array 
    [1, 2, 3] * Elementos 1, 2, 3

    * Object 
    { name: 'Fulano', idade: 20} *Elementos name: 'Fulano', idade: 20
